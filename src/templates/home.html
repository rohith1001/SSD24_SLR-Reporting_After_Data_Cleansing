Heatmap

This heatmap represents null values in every column(null values represented as yellow patches). If too many null values exist in a column, then it is better to drop that column, as it would not be contributing to our analysis.

Monthwise

This bar graph on the data sample shows the trend in number of papers published for every month. If month field is not specified, it would be shown as 'none'.

Yearwise

This bar graph depicts the trends in number of papers published each year. The range of years considered here depends on the data sample.

Piechart

This pie chart shows the top 5 keywords that occured in all research papers. It gives information about the top 5 research area trends.

Number of pages

This count plot shows the number of pages in a research paper and count of such research papers. This helps us study the trend in number of pages in each research paper.

Number of publications

This graph depicts the number of publications corresponding to a publisher. For example, we could have many publishers that have only one publication.

Keyword counts

This graph shows the number of occurrences of different keywords. For example, we can have some keywords with multiple occurrences and some keywords with only single occurrence.
